1 - Criar um projeto utilizando JQuery-UI

2 - Incluir uma página WEB contendo um formulário com dois campos - nome e data de nascimento - e um botão INCLUIR.

3 - A página também deve apresentar uma área para apresentar os dados, conforme item 4 e um textarea para apresentar o objeto JSON, conforme item 4.3

4 - Ao clicar no botão INCLUIR:

  • criar um script que leia os dados de nome e data de nascimento do formulário e inclua em um objeto JSON.
  • criar uma estrutura HTML (pode ser tabela) que apresente os dados em tela.
  • apresentar no textarea a estrutura do JSON atualizado (um vetor de JSON).

5 - Validar os campos do formulário da seguinte forma:

  • o campo nome não pode ser vazio e deve ter entre 3 e 50 caracteres.
  • o campo data não pode aceitar valores para menores de 18 anos, isto é, a data de nascimento deve considerar esse requisito.

6 - Apresentar o componente dialog para apresentar mensagem de erro.
